The namespace in which everything within Dave's Code Library is stored inside.

To prevent name clashes between other libraries/code which any application using DCL may include.

◆ sigmoid() [2/2]

double DCL::sigmoid ( double dValue,
double dResponse = 1 )
inline

Sigmoid function.

Parameters
dValueThe value to position upon an S curve
dResponseThe shape of the S curve
Returns
The computed value

Given a value, takes that and scales it so that the values lie upon an S curve. dResponse curve specified the shape of the curve. Higher values, flatten the curve, lower ones tighten it. A default value of 1 is usually used. Do not set it to zero, this'll create a divide by zero error.
