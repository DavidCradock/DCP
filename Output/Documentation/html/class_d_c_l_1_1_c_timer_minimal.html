Detailed Description

Deals with time related concepts with minimal features compared to CTimer.

Create an object of this class, keep it around and call update() between the code which you wish to measure the time difference. Then call getSecondsPast() to obtain the time passed since the two calls to update() This class is similar to CTimer, but with less features and is slightly faster to compute. It also returns the time delta as a double instead of float for more precision.
