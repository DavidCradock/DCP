Detailed Description

Master include header file for the entire DCL library meant to be included by any applications which wish to access all the code inside DCL.

If we wish to use the code inside DCL, then when creating an executable, include this header file which includes all code of DCL. Then, link to the compiled DavesCodeLibrary.lib which can be found located at /x64/Release/DavesCodeLibrary.lib
