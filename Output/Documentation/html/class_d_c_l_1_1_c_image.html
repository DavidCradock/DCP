Detailed Description

A class for creating/loading/saving/modifying 2D images.

Can read the following formats... JPEG baseline & progressive (12 bpc/arithmetic not supported, same as stock IJG lib) PNG 1 / 2 / 4 / 8 / 16 - bit - per - channel TGA(not sure what subset, if a subset) BMP non - 1bpp, non - RLE PSD(composited view only, no extra channels, 8 / 16 bit - per - channel) GIF(*comp always reports as 4 - channel) HDR(radiance rgbE format) PIC(Softimage PIC) PNM(PPM and PGM binary only) Image pixels are stored in row first, then column. unsigned int iPixelIndex = iPixelPosX + (iPixelPosY * _miWidth);

void DCL::CImage::normalmap ( CImage & outputImage,

Computes a normal map used for normal mapping from this image and stores the result in outputImage.

Parameters
outputImageThe image which will hold the normal map
fScaleValue between 0.0 to 1.0 to signify how "intense" the normals are.

This image should be a heightmap, where each pixel represents the height of a surface. White being max height, black being the lowest. However, this image first creates a copy of itself in memory, then calls greyscaleSimple() on that to ensure proper computation of the normals. fScale should be between 0.0f and 1.0f and affects how "intense" the normals are generated. This value is clamped internally. Lower values increase the effect If this image contains no data, an exception occurs.
