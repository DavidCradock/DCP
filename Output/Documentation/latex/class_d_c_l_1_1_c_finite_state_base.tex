\doxysubsection{Detailed Description}
Finite state base class with which to derive our FSM state classes from for use with the \doxylink{class_d_c_l_1_1_c_finite_state_machine}{CFinite\+State\+Machine} class. 

This represents a single state in a finite state machine and has several pure virtual methods which need to be implemented within any class which is derived from this one.
