We provide a vector of strings holding each individual image\textquotesingle{}s filename. Each individual image can have different dimensions. We specify a maximum width and height that an individual atlas image may be. We specify whether to allow or disallow rotation of images to allow better fitting of the images. If rotation is enabled, the computed texture coordinates are rotated accordingly. After calling this, we can call the various other methods to get at the information If the vector of strings is empty, an exception occurs If an individual image\textquotesingle{}s dimensions, including spacing is greater than the max atlas dimensions, an exception occurs. If all given images are the same dimensions, nothing is sorted.
