\doxysubsection{Detailed Description}
A class for creating/loading/saving/modifying 2D images. 

Can read the following formats... JPEG baseline \& progressive (12 bpc/arithmetic not supported, same as stock IJG lib) PNG 1 / 2 / 4 / 8 / 16 -\/ bit -\/ per -\/ channel TGA(not sure what subset, if a subset) BMP non -\/ 1bpp, non -\/ RLE PSD(composited view only, no extra channels, 8 / 16 bit -\/ per -\/ channel) GIF(\texorpdfstring{$\ast$}{*}comp always reports as 4 -\/ channel) HDR(radiance rgb\+E format) PIC(\+Softimage PIC) PNM(\+PPM and PGM binary only) Image pixels are stored in row first, then column. unsigned int i\+Pixel\+Index = i\+Pixel\+PosX + (i\+Pixel\+PosY \texorpdfstring{$\ast$}{*} \+\_\+mi\+Width);
