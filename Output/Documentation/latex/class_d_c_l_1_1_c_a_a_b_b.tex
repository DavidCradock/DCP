\doxysubsection{Detailed Description}
An Axis Aligned Bounding Box. 

An AABB is simply a box with it\textquotesingle{}s axis aligned to the world. It is similar to the 2D \doxylink{class_d_c_l_1_1_c_rect}{CRect} class, but in 3D and is used for culling with the \doxylink{class_d_c_l_1_1_c_oct_tree}{COct\+Tree} class
