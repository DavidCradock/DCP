\doxysubsection{Detailed Description}
Deals with time related concepts with additional features compared to \doxylink{class_d_c_l_1_1_c_timer_minimal}{CTimer\+Minimal}. 

Create an object of this class, keep it around and call \doxylink{class_d_c_l_1_1_c_timer_a1c746ac5f8bf4032562a785dda02eb39}{update()} between the code which you wish to measure the time difference. Then call \doxylink{class_d_c_l_1_1_c_timer_accc8d2378a3e86ffe1c74dbc79c454b4}{get\+Seconds\+Past()} to obtain the time passed since the two calls to \doxylink{class_d_c_l_1_1_c_timer_a1c746ac5f8bf4032562a785dda02eb39}{update()} There are also two framerate methods which return what the framerate is, based on the time passed between the last two calls to \doxylink{class_d_c_l_1_1_c_timer_a1c746ac5f8bf4032562a785dda02eb39}{update()}. There\textquotesingle{}s \doxylink{class_d_c_l_1_1_c_timer_a9a06de522d2c363ea7d046d423cedb50}{get\+FPS()} which simply returns the current frames per second and then there\textquotesingle{}s... \doxylink{class_d_c_l_1_1_c_timer_abda1c0dcff3b3a9877320da1b6241228}{get\+FPSAveraged()} which returns the frames per second which have been averaged over a certain amount of time. This "{}certain amount of time"{} is, by default, set to be 1 update per second, but can be changed with a call to \doxylink{class_d_c_l_1_1_c_timer_ab1e55329385452b5a1b604995353912b}{set\+Averaged\+FPSRate()}
