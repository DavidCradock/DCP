\doxysubsection{Detailed Description}
Deals with time related concepts with minimal features compared to \doxylink{class_d_c_l_1_1_c_timer}{CTimer}. 

Create an object of this class, keep it around and call \doxylink{class_d_c_l_1_1_c_timer_minimal_a456ee8e96851304a935a60bbd1e1f175}{update()} between the code which you wish to measure the time difference. Then call \doxylink{class_d_c_l_1_1_c_timer_minimal_ac90e8e8d0853009cda8dd832587e7da6}{get\+Seconds\+Past()} to obtain the time passed since the two calls to \doxylink{class_d_c_l_1_1_c_timer_minimal_a456ee8e96851304a935a60bbd1e1f175}{update()} This class is similar to \doxylink{class_d_c_l_1_1_c_timer}{CTimer}, but with less features and is slightly faster to compute. It also returns the time delta as a double instead of float for more precision.
