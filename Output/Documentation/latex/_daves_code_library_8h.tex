Master include header file for the entire \doxylink{namespace_d_c_l}{DCL} library meant to be included by any applications which wish to access all the code inside \doxylink{namespace_d_c_l}{DCL}. 

If we wish to use the code inside \doxylink{namespace_d_c_l}{DCL}, then when creating an executable, include this header file which includes all code of \doxylink{namespace_d_c_l}{DCL}. Then, link to the compiled Daves\+Code\+Library.\+lib which can be found located at /x64/\+Release/\+Daves\+Code\+Library.lib
